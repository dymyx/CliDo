# ToDoCLI

Простое CLI-приложение для управления вашим списком дел.

## Особенности

- Добавление, удаление, редактирование и просмотр задач.
- Работа с несколькими файлами списков задач.
- Интерактивные подсказки для удобного использования.
- Настройка каталога для хранения файлов с задачами.

## Установка

1.  Клонируйте репозиторий:
    ```bash
    git clone <URL-вашего-репозитория>
    cd ToDoCLI
    ```
2.  Установите зависимости:
    ```bash
    npm install
    ```
3.  Соберите проект:
    ```bash
    npm run build
    ```
4.  Для глобального использования создайте символическую ссылку:
    `bash
npm link
`
    После этого вы сможете использовать команду `todo` из любого места в вашей системе.

## Использование

### Основные команды

- `todo look` - Просмотр задач из выбранного файла.
- `todo add` - Добавление новой задачи.
- `todo done` - Отметить задачу как выполненную/невыполненную.
- `todo edit` - Редактировать задачу (название, описание, статус).
- `todo remove` - Удалить задачу.
- `todo path <путь>` - Установить каталог для хранения файлов с задачами.

### Пример работы

1.  **Настройка каталога**

    Прежде всего, укажите, где вы хотите хранить файлы с задачами.

    ```bash
    todo path ./my-tasks
    ```

    Команда создаст папку `my-tasks` в текущем каталоге, если она не существует.

2.  **Добавление задачи**

    ```bash
    todo add
    ```

    Приложение предложит выбрать файл (или создать новый, если файлов нет) и затем попросит ввести название и описание для новой задачи.

3.  **Просмотр задач**

    ```bash
    todo look
    ```

    Выберите файл, и вы увидите список всех задач в нем с их статусами.

4.  **Изменение статуса задачи**

    ```bash
    todo done
    ```

    Выберите файл, а затем задачу, статус которой вы хотите изменить. Статус переключится на противоположный (Выполнена <-> Не выполнена).

5.  **Редактирование задачи**

    ```bash
    todo edit
    ```

    Выберите файл и задачу для редактирования. Вам будет предложено ввести новое название, описание и установить статус выполнения.

6.  **Удаление задачи**

    ```bash
    todo remove
    ```

    Выберите файл и задачу, которую хотите удалить. Потребуется подтверждение.
